# Natural Language to SQL Query App (Gemini + Streamlit)

This project converts plain English questions into SQL queries using **Google Gemini (2.5 Flash)**.
The generated SQL is executed on a local **SQLite** database, and the results are displayed through a clean **Streamlit** interface.

This project is perfect for beginners, data analysts, and anyone exploring **AI + SQL automation**.
